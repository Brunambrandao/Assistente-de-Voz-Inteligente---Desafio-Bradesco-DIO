# Assistente-de-Voz-Inteligente---Desafio-Bradesco-DIO
Assistente de voz Personal Banker desenvolvido para o Bootcamp Bradesco/DIO. Integra OpenAI Whisper, Google Gemini e gTTS para um ciclo completo de voz. O diferencial técnico é a implementação de uma lógica de fallback para garantir alta disponibilidade e continuidade do atendimento, mesmo em caso de falhas em APIs externas.
